逍遥人才网
主要功能： 
1、园区和开发区展示：中国各省市工业园区和开发区展示，介绍。
2、政策展示：中国各地区招商引资政策、人才政策及其他政策发布。 
3、创业项目发布：发布各地区的创业人才项目，供创业人才申报。 
4、工作机会：发布企业和科研院所工作机会。
5、会员注册系统：人才可以注册平台成为平台用户，成为会员可以申报创业项目，申请工作岗位
6、项目申报系统：人才申报项目，需要填写详细的个人信息，教育背景，当前所在地（国内和海外），职业履历，研究方向和研究成果。并提交详细的个人简历和创业计划书。 
7、职位申报系统：人才申报职位，需要填写详细的个人信息，教育背景，当前所在地（国内和海外），职业履历，研究方向和研究成果。并提交详细的个人简历。 
8、后台管理系统：配置系统管理员具有超级权限，系统管理员可添加、删除管理员；管理员编辑、增删各省市的园区和开发区；管理员编辑、增删国家和各省市的人才政策、招商引资政策以及其他政策；管理员可审核用户注册信息；管理员审核会员项目申报信息，收集会员简历和创业计划书。
9、能够容纳1000人同时在线。
10、跨平台展示，包括：pc端、手机端、平板端
